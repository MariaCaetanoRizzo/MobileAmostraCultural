# MobileAmostraCultural
- Repositório para mandar e criar os projetos da amostra
- Alunos produtores do projeto: Arthur Paixão Serafim, Maria Caetano Rizzo, Rihan de Jesus
- Nosso projeto foca em mostrar um pouco de tudo sobre o Chile, tanto sua história e folclore quanto os pontos turísticos do Chile.
